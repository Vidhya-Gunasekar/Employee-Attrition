📌 Project Title

HR Analytics & Employee Attrition Dashboard using Power BI

📖 Project Overview

This project presents an interactive HR Analytics Dashboard developed in Power BI to analyze employee attrition patterns and workforce trends. The dashboard helps HR teams understand why employees leave, how salary and experience affect retention, and which departments require attention.

It converts employee data into meaningful visual insights that support better workforce planning and retention strategies.

🎯 Project Objectives
Analyze employee attrition trends across departments
Calculate Attrition Percentage
Evaluate Average Salary distribution
Measure Employee Tenure
Study hiring patterns across experience levels
Compare attrition by gender and department
Provide actionable insights to improve retention strategies

🛠 Tools & Technologies Used
Microsoft Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Data Modeling
Interactive Dashboard Visualizations

📂 Dataset Description

The dataset contains employee-related information such as:

Employee demographics
Department details
Salary structure
Experience level
Tenure
Attrition status
Gender distribution
Hiring trends

This data is used to identify workforce behavior and attrition risk factors.

📊 Key Dashboard Features

The dashboard includes:

Attrition % calculation using DAX 📉
Average Salary analysis 💰
Employee Tenure tracking ⏳
Department-wise attrition comparison 🏢
Gender-based workforce insights 👩‍💼👨‍💼
Experience-level hiring trends 📈
Interactive filters for:
Department
Gender
Experience

These filters allow dynamic and customized analysis.

📐 Calculated Measures Used

The following important calculated measures were implemented:

Attrition % = (Employees Left / Total Employees) × 100
Average Salary
Average Employee Tenure

These measures help identify retention patterns and workforce stability.

🔍 Key Insights Generated

From the dashboard analysis:

Departments with higher attrition rates can be identified
Salary trends affecting employee retention become visible
Experience-level hiring patterns are clearly understood
Gender-based workforce distribution can be analyzed
Tenure insights help detect early-stage employee exits

These insights support data-driven HR decision-making.
